CSS

Layout Systems

Visual Design

  • Typography, line-height, and readability
  • Color systems and contrast
  • Shadows, borders, and background styles

Responsive and Motion

  • Media queries for different screens
  • Transitions for smooth changes
  • Keyframe animations for storytelling

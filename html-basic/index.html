HTML

HTML is the language of structure. It defines the content and meaning of every page element so browsers, search engines, and assistive tools can understand it.

Core Building Blocks

  • Elements and tags: <div>, <section>, <button>
  • Attributes: href, src, alt, id, class
  • Document structure: <head>, <body>

Semantic HTML

  • Meaningful layout with <header>, <main>, <footer>
  • Improves SEO and accessibility
  • Makes code readable for teams

Forms and Inputs

  • Collect user data with <form>
  • Input types like email, password, date
  • Labels and validation essentials

Media and Accessibility

  • Images, audio, and video embeds
  • Use alt text and proper headings
  • Keyboard-friendly structure

CSS Layout Basics

Use HTML structure + CSS layout properties to build clean sections.

HTML Tags

  • <section> Ś groups a content block.
  • <h2> Ś section heading.
  • <p> Ś description text.
  • <div> Ś layout wrapper.

CSS Properties

  • display: grid Ś make a grid layout.
  • gap Ś space between items.
  • padding Ś inner spacing.
  • border-radius Ś rounded corners.

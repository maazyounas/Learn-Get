CSS Layout Basics

Use HTML structure + CSS layout properties to build clean sections.

HTML Tags

  • <section> — groups a content block.
  • <h2> — section heading.
  • <p> — description text.
  • <div> — layout wrapper.

CSS Properties

  • display: grid — make a grid layout.
  • gap — space between items.
  • padding — inner spacing.
  • border-radius — rounded corners.

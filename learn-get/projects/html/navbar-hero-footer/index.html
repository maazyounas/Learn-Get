Build a complete layout: Navbar, Hero, Footer

Learn semantic HTML by editing real code. Every change reflects instantly in the preview. Understand why structure matters.

Key tags you will use

  • <header> - wraps top banner and hero.
  • <nav> - main navigation links.
  • <section> - groups hero or content blocks.
  • <h1> - primary headline (only one per page).
  • <footer> - copyright and extra info.
  • <ul> / <li> - navigation menus.
Semantic = accessible + SEO

What to do

  1. Edit the HTML in the editor below.
  2. Preview updates automatically (right side).
  3. Change the brand name, add links, rewrite the hero text.
  4. Hit Reset to restore the starter layout.

Tip: Try adding a second button or a background color inside the hero.

Why this matters

Every website uses these three pillars: navigation (navbar), main message (hero), and closure (footer). Mastering them builds a solid foundation for responsive design and user experience.

Pro: Use display: flex in your navbar to align items horizontally.
